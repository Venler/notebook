# Javascript查找

## 二位数组中查找

```js
function find(array,target){
    var h = array.length;
    var w = array[0].length;
    if(h === 0){
        return false;
    }
    var i = 0;
    var j = h-1;
    while(i <= w-1 && j >= 0){
        if(array[j][i] > target){
            j--;
        }
        else if(array[j][i] < target){
            i++;
        }
        else{
            return true;
        }
    }
    if(i > w-1 || j < 0){
        return false;
    }
}
```
